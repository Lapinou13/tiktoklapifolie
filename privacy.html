Politique de confidentialité

Dernière mise à jour : 30 juin 2026

Cette application ("TikTok App") est un outil personnel développé par Romann Rousseau pour publier automatiquement du contenu sur son propre compte TikTok via l'API officielle TikTok for Developers.

Données collectées

Cette application utilise l'authentification OAuth 2.0 fournie par TikTok pour obtenir l'autorisation de publier du contenu sur le compte TikTok de son propriétaire. Aucune donnée personnelle d'autres utilisateurs n'est collectée, stockée ou partagée.

Utilisation des données

Les jetons d'accès (access tokens) fournis par TikTok sont utilisés uniquement pour publier du contenu vidéo sur le compte autorisé. Ils ne sont transmis à aucun tiers.

Stockage

Les identifiants d'accès sont stockés de manière sécurisée et utilisés uniquement dans le cadre du fonctionnement de cette application personnelle.
